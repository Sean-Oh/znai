# Pie

Given a `CSV` file

:include-file: data.csv

To build a pie chart use

    :include-chart: data.csv {type: "Pie"}

:include-chart: data.csv {type: "Pie"}

To change inner radius use

    :include-chart: data.csv {type: "Pie", innerRadius: 100} 

:include-chart: data.csv {type: "Pie", innerRadius: 100}

# Bar
