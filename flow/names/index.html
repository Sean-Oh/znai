Automatic Names

By default, page names are automatically derived from file names. file-name.md becomes File Name page title.

Name Overrides

To override the default, add the following syntax to the top of your Markdown file: --- title: Custom Name --- To keep things easy for future documentation owners, it's good practice to keep page and file names the same.
